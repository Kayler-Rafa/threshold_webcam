# Threshold com OpenCV

Este projeto demonstra o funcionamento do **threshold (limiar)** em visão computacional usando **Python e OpenCV**.

O programa captura a imagem da **webcam em tempo real**, converte para **escala de cinza** e aplica um **threshold binário**, transformando a imagem em apenas **preto e branco**.

Um **controle deslizante (trackbar)** permite alterar o valor do limiar em tempo real e observar como os pixels são classificados.

Também é exibido um **histograma dinâmico**, mostrando a distribuição das intensidades dos pixels e a posição atual do threshold.

Pressione **ESC** para encerrar o programa.
